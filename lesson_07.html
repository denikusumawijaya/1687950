Spiral Review Warm-up

00:00 - 00:10

Before we start, let's review the basics of algebraic expressions. / Sebelum memulai, mari kita review dasar ekspresi aljabar.

An algebraic expression contains variables (unknown numbers, like \(x\) or \(y\)) and constants (fixed numbers, like \(-7\) or \(4\)). / Ekspresi aljabar mengandung variabel (angka tak diketahui) dan konstanta (angka tetap).

We can simplify expressions by combining like terms. For example, \(3y + 4 - y = 2y + 4\). / Kita bisa menyederhanakan ekspresi dengan menggabungkan suku sejenis.

Concept of Balance

00:10 - 00:15

A linear equation is like a prehistoric balance scale. Whatever you put on one side, you must put on the other to keep it balanced. / Persamaan linear seperti timbangan prasejarah. Apa yang ditambah di kiri, harus ditambah di kanan.

If the scale tips, the equation is no longer equal! / Jika timbangan miring, persamaan tidak seimbang!

Game: Keep it Balanced! / Jaga Keseimbangan!

If you add 5 to the left side, what must you do to the right side? Drag the correct action. / Jika tambah 5 di kiri, apa yang harus dilakukan di kanan? Seret aksi yang benar.

Inverse Operations

00:15 - 00:20

To solve an equation, we use inverse operations (operasi balikan) to isolate the variable. It's like untying a knot! / Untuk menyelesaikan persamaan, gunakan operasi balikan untuk mengisolasi variabel. Seperti membuka simpul!

The inverse of addition is subtraction, and the inverse of multiplication is division. / Lawan dari penjumlahan adalah pengurangan, lawan dari perkalian adalah pembagian.

Operation Inverse
Addition (+) / Penjumlahan Subtraction (-) / Pengurangan
Multiplication (\times) / Perkalian Division (\div) / Pembagian

Game: Match the Inverse / Cocokkan Lawannya

Drag the correct inverse operation to the box. / Seret operasi balikan yang benar ke kotak.

Unwrapping the Variable

00:20 - 00:30

When solving \(3x + 5 = 20\), think of \(x\) as a present. First, it's wrapped in a box (multiplied by 3), then put in a bag (added 5). / Pikirkan x seperti hadiah. Pertama dibungkus kotak (dikali 3), lalu dimasukkan tas (ditambah 5).

To get the present, you must open the bag first (subtract 5), then open the box (divide by 3). Always undo the outer layer first! / Untuk mendapat hadiah, buka tas dulu (kurangi 5), lalu buka kotak (bagi 3). Selalu buka lapisan luar duluan!

This is called performing operations in the reverse order of PEMDAS/BODMAS. / Ini disebut melakukan operasi secara terbalik dari aturan PEMDAS.

Game: Order the Steps / Urutkan Langkah

Guided Practice

00:30 - 00:45

Let's apply our unwrapping technique! Remember, move the constant first, then the coefficient. / Mari terapkan teknik membuka bungkus! Pindahkan konstanta dulu, lalu koefisien.

If variables are on both sides (e.g., \(5x + 2 = 2x + 11\)), group them on one side first! / Jika variabel di kedua sisi, kumpulkan dulu di satu sisi!

Worked Example / Contoh Terbimbing

Step 1:
\(5x + 2 = 2x + 11\)
\(-2x\) from both sides
Step 2:
\(3x + 2 = 11\)
\(-2\) from both sides
Step 3:
\(3x = 9\)
\(\div 3\) on both sides
Result:
\(x = 3\)
