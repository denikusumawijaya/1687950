Introduction & Real-World
00:10 - 00:15

Imagine a sophisticated barcode scanner in a factory. When you scan a barcode (input), the machine MUST ONLY produce one price (output). If one barcode gives two different prices, the system is broken!

Bayangkan mesin scanner barcode di pabrik. Saat scan barcode (input), mesin HANYA BOLEH mengeluarkan satu harga (output). Jika satu barcode keluar dua harga, sistem kacau!

In mathematics, the rule of 'one input produces exactly one output' is called a FUNCTION. A relation that breaks this rule is NOT a function.

Dalam matematika, aturan 'satu input menghasilkan tepat satu output' disebut FUNGSI. Relasi yang melanggar aturan ini BUKAN fungsi.

Fungsi: 1 Input → 1 Output yang tepat

Set Identification Quiz (10 Items)

Problem Introduction
00:20 - 00:30

A FUNCTION is a special relation where EVERY member of the starting set (Domain) maps to EXACTLY ONE member of the target set. If one input branches out to two or more outputs, the machine is broken, and it is ONLY a RELATION.

FUNGSI adalah relasi khusus di mana SETIAP anggota himpunan awal (Domain) memetakan TEPAT SATU anggota himpunan tujuan. Jika satu input bercabang ke dua output atau lebih, mesinnya rusak, dan itu HANYA RELASI.

NOT a Function

1 Input punya 2 Output!

FUNCTION

1 Input = 1 Output!

Key Terms / Istilah Kunci:

  • Domain (Daerah Asal): The starting set (Inputs). Himpunan awal (Input).
  • Codomain (Daerah Kawan): The target set (All possible outputs). Himpunan tujuan.
  • Range (Daerah Hasil): The actual outputs that get hit by arrows. Output yang terkena panah.

Domain, Codomain, Range & Function Check (10 Items)

Guided Practice
00:30 - 00:45

Now we enter the hard level! We use the notation f(x) which means "function of x". Think of it as a machine instruction. If $f(x) = 2x + 3$, it means: "Take the input, multiply by 2, then add 3".

Sekarang kita masuk level sulit! Kita gunakan notasi f(x) yang berarti "fungsi dari x". Anggap saja ini instruksi mesin. Jika $f(x) = 2x + 3$, artinya: "Ambil input, kalikan 2, lalu tambah 3".

Mesin Pabrik Fungsi / Function Factory Machine

Example: Find $f(5)$ if $f(x) = 2x + 3$. Replace $x$ with 5 $\rightarrow f(5) = 2(5) + 3 = 10 + 3 = 13$.

Contoh: Hitung $f(5)$ jika $f(x) = 2x + 3$. Ganti $x$ dengan 5 $\rightarrow f(5) = 2(5) + 3 = 10 + 3 = 13$.
